# 绿色足迹 🌱

一个基于 Vue3 + Vite 的环保行动记录与互动 Web 项目，帮助用户记录环保行动，参与社群挑战，共同守护地球家园。

## 📋 项目概述

「绿色足迹」致力于通过数字化方式鼓励和记录用户的环保行为，包括垃圾分类、绿色出行、节能减排等多种环保行动。用户可以通过记录日常环保行为获得积分，参与社群挑战，解锁环保勋章，培养可持续的环保习惯。

## 🚀 功能特色

### 🏠 首页
- **导航栏**：响应式设计，包含用户登录状态显示
- **核心功能入口**：快速访问各个主要功能模块
- **个人贡献展示**：展示用户的环保行动记录和成就
- **热门挑战**：展示当前最热门的环保挑战活动
- **精选知识**：提供环保知识普及和学习资源

### 📝 行动记录页
- **行动类型选择**：支持垃圾分类、绿色出行、节约用水、节约能源、植树造林、环境清洁等多种类型
- **智能记录表单**：根据行动类型动态调整表单字段
- **图片上传**：支持上传环保行动现场照片
- **贡献值预览**：实时计算和显示行动可获得积分
- **时长加成**：根据行动持续时间提供额外积分奖励

### 🌍 足迹数据页
- **数据统计栏**：展示总贡献值、行动次数、虚拟绿植数量、用户等级等核心数据
- **动态足迹图谱**：可视化展示用户环保行动的地理分布
- **虚拟绿植区**：
  - 虚拟植物养成系统
  - 浇水互动功能
  - 绿植商店兑换系统
- **行动统计分析**：
  - 行动类型分布图表
  - 每日行动趋势分析
- **成就徽章系统**：展示已获得和未解锁的徽章

### 🏆 社群挑战页
- **功能标签**：按类型筛选挑战活动
- **挑战内容卡片**：详细展示挑战信息、参与人数、奖励等
- **排行榜系统**：日榜、周榜、月榜多维度排名
- **挑战创建**：用户可自定义创建环保挑战
- **进度追踪**：实时显示挑战完成进度

### 🔧 组件模块

#### 登录注册组件
- 用户认证系统
- 用户信息管理
- 头像上传功能
- 登录状态持久化

#### 垃圾分类查询组件
- **智能搜索**：支持关键词搜索垃圾分类信息
- **分类展示**：可回收物、有害垃圾、湿垃圾、干垃圾四大分类
- **详细信息**：每种垃圾的处理方法和环保提示
- **快速筛选**：按分类类型快速查看常见物品

#### 环保场景库组件
- **场景分类**：家庭、办公、校园、社区等多种环保场景
- **步骤指导**：详细的环保行动实施步骤
- **环保效益**：量化展示环保行动的环境效益
- **相关行动推荐**：根据场景推荐相应的环保行动

#### 勋章解锁组件
- **勋章分类**：行动类、挑战类、特殊类、里程碑等多种类型
- **稀有度系统**：普通、稀有、罕见、史诗、传说五个等级
- **进度追踪**：实时显示各勋章的完成进度
- **解锁动画**：精美的勋章解锁动画效果
- **分享功能**：支持将获得的勋章分享到社交媒体

## 🛠️ 技术栈

- **前端框架**：Vue 3.x
- **构建工具**：Vite
- **路由管理**：Vue Router
- **状态管理**：Pinia/Vue3 Reactive
- **样式方案**：CSS3 + Scoped CSS
- **图标系统**：Emoji + 自定义图标
- **响应式设计**：移动端适配
